Plutosoft Delimited : Batch processing audio speech to text.

Sat 22 April 2017

Recognize speech, not wreck a nice beach

People are starting to warm up to speech-to-text tools available from commercial off the shelf application software, text editors in desktop, notepads and email clients in mobile operating systems, and cloud-based document editors. But what if you have a large-ish number of audio files? It turns out there isn't an easy way to handle this using those tools, and the commercially available alternatives are human labor intensive, and therefore, rather unaffordable to the typical consumer who just happens to have a boat load of such files.

Fortunately, an affordable alternative is emerging, if you're willing to write a bit of code. The hyperscale cloud api providers are making their deep learning models available for a small fraction of the dollar cost, and able to handle large batches in a single bound, although there is that small issue of having to write some code in order to get this amazing cost and time reduction.

Although generally available, there is a ways to go before these APIs are generally useful for converting noisy audio recorded in the wild using mobile phones. Nonetheless, the results are good enough and the pricing low enough that it may time to dip your metaphorical toes in the water.
